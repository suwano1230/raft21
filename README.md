# raft21（ソクタイプ型のDIYキーボード）
## 概要
Ploverでの使用を前提にしたStenturaプロトコル互換で動くソクタイプ型のDIYキーボードです。
ソクタイプは21キーで、キーの配置やキーキャップのサイズ、表面のくぼみ、高低差があるなどの特徴があります。
Ploverでの動作は確認できましたが、本家Stenturaプロトコルと同一のものではありません。
実機などから情報を得てStenturaプロトコルを再現したということはありません。
また、キーキャップのサイズなどに関して、ソクタイプの厳密な再現とは言えない可能性があります。
## PCB基盤とスイッチプレート
KiCadを使用して、PCB基盤とスイッチプレートを作成します。MCUはRP2040（Raspberry Pi Pico）を使用します。
## キーキャップ
ソクタイプ型のキーキャップを再現するために作成します。
## ケース
通常、自作キーボードを作成するときと同様に作成します。
## キースイッチ
市販のCherry MX互換スイッチを使用します。赤軸静音の使用を推奨します。
