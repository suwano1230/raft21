# raft21（ソクタイプ型のDIYキーボード）
## 概要
Ploverでの使用を前提にしたStenturaプロトコルで動くキーボードをDIYするためのものです。
ソクタイプは21キーで、キーの配置やキーキャップのサイズ、表面のくぼみなどに特徴があり、それを再現することが目的です。
## PCB基盤とスイッチプレート
KiCadを使用して、PCB基盤とスイッチプレートを作成します。
## キーキャップ
OpenSCADを使用して、ソクタイプ型のキーキャップを再現するために作成します。
## キースイッチ
市販のCherry MX互換スイッチを使用します。
## ケース
OpenSCADを使用して作成します。
