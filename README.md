# raft21（ソクタイプ型のDIYキーボード）
## 概要
Ploverでの使用を前提にしたStenturaプロトコルで動くDIYキーボードです。
ソクタイプは21キーで、キーの配置やキーキャップのサイズ、表面のくぼみなどに特徴があります。
## PCB基盤とスイッチプレート
KiCadを使用して、PCB基盤とスイッチプレートを作成します。MCUはRP2040を使用します。
## キーキャップ
OpenSCADを使用して、ソクタイプ型のキーキャップを再現するために作成します。
## キースイッチ
市販のCherry MX互換スイッチを使用します。
## ケース
OpenSCADを使用して作成します。
